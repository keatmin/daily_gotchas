Using IAM to connect to RDS

August 19, 2021 Â· One min read
keatmin
Data nerd
  1. Enable IAM DB authentication for the database
  2. Create IAM user that maps to the database user via
CREATE USER foo with login;GRANT rds_iam to foo;
  1. Attach role to user/ec2
rds-db:connectresources: arn:dbuser/foo
  1. Generate AWS authentication token assuming AWS CLI profile name is bar
export RDSHOST=rds.dns.comexport PGPASSWORD="$(aws rds generate-db-auth-token                         --hostname $RDSHOST                        --port 5432                        --region <region_name>                        --username foo                        --profile bar)"
  1. Connect with
AWS_PROFILE=bar psql "host=$RDSHOST port=5432 dbname=dbname user=foo password=$PGPASSWORD"
